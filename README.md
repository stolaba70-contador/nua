# RT 59 — NUA Interactiva

App interactiva para navegar la Resolución Técnica N°  59 (T.O. RT 54 — Norma Unificada Argentina de Contabilidad).

## Características

- 📖 **1.078 párrafos** organizados en 11 capítulos + Apéndice A
- 🔍 **Búsqueda instantánea** por número de párrafo o palabra clave
- 🔗 **Referencias cruzadas clickeables** entre párrafos
- 📚 **Glosario** con 88 términos definidos
- 📱 **Responsive** — funciona en desktop y mobile
- ⚡ **Sin backend** — HTML standalone, carga instantánea

## Deploy

El sitio se publica automáticamente con GitHub Pages desde la rama `main`.
